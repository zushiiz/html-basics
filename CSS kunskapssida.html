CSS

Vad är CSS? Hur används det?

CSS (Cascading Style Sheet) används för hemsidans utseende.

Vad är skillnaden mellan inline-stil, inbäddad stil och extern stil i CSS?

Inbäddad css används i html-filen genom en style tag. Inline är när man skriver in css inne i ett enda specifikt tag. Extern är css i en seperat css-fil. Man länkar css-filen till html-filen. Extern är bättre om det ska ha css på flera sidor medan inbäddad är bra för en enda sida.

Vad är en selektor i CSS och hur används den?

Används för att tillämpa stilar till specifika html element. Det finns primärt fem sorters sektrorer. Enkel - väljer från namn. Kombinerade - väljer utifrån specifikt förhållande mellan element. T.ex. child till något parent element. Pseudoklass - Väljer utifrån vad som händer med ett element t.ex. hover. Pseudoelement - väljer en specifikt del av ett element Attribut - väljer från attribut.

Vad är skillnaden mellan klass-selektor och ID-selektor i CSS?

Id - Man ska aldrig använda mer än ett id/dokument. Ett specifikt tag som man vill ge ett namn till. Klass - Man kan använda på flera element. Namn - Man använder namnet av elementet för att ändra.

Vad är kaskadordningen i CSS och hur påverkar den hur stilar tillämpas på element?

Kaskadordningen styr hur stilar tillämpas på HTML-element när flera olika stilregler är i konflikt eller konkurrens med varandra. Kaskadordningen hjälper till att bestämma vilken stilregel som har företräde och därmed tillämpas på ett element.

Vad är boxmodellen i CSS och vilka element påverkas av den?

Alla HTML-element ses som boxar. Innehållet är innerst, omges av padding som i sin tur omges av border och sist av margin Padding - Frigör utrymme runt innehållet Margin - Frigör utrymme till andra boxar, utanför innehållet

Vad är "grid layout" och hur skiljer det sig från "flexbox"?

Grid layout används för komplicerade layouts. Den använder både rader och kolumner som ett koordinatsystem. Flexbox är enklare syntax än box model och behöver inte "float"/"clear" för att positionera.

Vad är skillnaden mellan positionering med "static", "relative", "absolute" och "fixed" i CSS?

Static - Standard. Påverkas inte av andra regler. Relative - Positionerar relativt till sitt ursprungliga läge, men påverkas av andra regler. Fixed - Fast position i förhållande till webbläsarfönstret och påverkas inte av scrollning. Lämnar inget utrymme på sidan där det "normalt" skulle varit placerat. Absolute - Positionerar relativt till närmaste föräldraelementet istället för till webbläsarfönstret. Om ingen förälder finns så positioneras elementet relativt och följer med scrollning.
